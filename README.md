Data Browser
===============================

version number: 0.1
author: roberttk

Overview
--------

Data browser plugin for xicam

Installation / Usage
--------------------

To install use pip:

    $ pip install Xi-cam.databrowser


Or clone the repo:

    $ git clone github.com/tangkong/Xi-cam.databrowser
    $ python setup.py install
